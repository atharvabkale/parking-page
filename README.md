# My Project

This small static site demonstrates a simple main menu page.

How to view
----------

Open `index.html` in your browser (double-click or run a local server).

Quick local server (Python):

```bash
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

Files
-----

- `index.html` — main menu and content sections
- `styles.css` — styling and responsive layout
- `script.js` — menu interactions and keyboard navigation

Feel free to edit sections and styles.
